Emmet Abbreviations in Visual Studio Code

Tutorial

In this short tutorial I demonstrate how to use "Emmet Abbreviations" in Visual Studio Code. Emmet Abbreviations help to speed up the way you write your HTML code. Emmet has many features however, I go over the basic use in HTML in this tutorial. Emmet comes with visual studio code by default, so there is no need for any extra installations. As well as HTML, Emmet also works with HAML, PUG, SLIM, JSX, XML, XSL, CSS, SCSS, SASS, LESS and STYLUS.
