This program isn't written using java code however, it is simulating how a java program would run in an IDE, such as eclipse.

Java Program Simulation

public class stringReverse
{

/**
* The Method reverses the word that is entered
* @param str
* @return the reverse of the word entered
*/
public static String reverse(String str)
{
String result = "";

for (int i = str.length()-1; i >= 0; i--)
{
result = result + str.charAt(i);
}
}

public static void main(String[] args)
{
String str = reverse("");

System.out.println(str);
}
}
Try it yourself!
  • Press "Clear" to clear the existing word.
  • Type the word you would like to reverse and press "Enter".
  • Press "Run" to run the program.
