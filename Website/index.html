About Me

Hi. My name is Liam. I am nineteen years old from Lanesboro Co. Longford. I am currently a first year student studying Software Design with Virtual reality and gaming (AL801) at Athlone's Institute of Technology. I developed a strong interest for computers during my time in secondary school. I was curious as to how computers actually function both from the hardware and software side of things. This lead to me building my first computer back in 2015, which is the machine I currently use today. I've also had a love for gaming since a young age, starting from a playstation 2 and progressing to PC gaming. All of this lead to me choosing this course and as I prefer the software aspect of computers over the hardware, I knew it was perfect.

My Skills

  • Web Development

    Web Development is one of my favourite modules that I am currently studying. I have had a great interest in building websites since transition year of secondary school. I learned the basics of HTML and CSS during my time in transition year, which was a big help coming into college. I have learned a lot more since entering college and I am able to create a fully functioning, structured websites using HTML and CSS. The third component to web development after HTML and CSS is JavaScript, which I had no prior experience with. I am currently trying to put most of my focus towards JavaScript as it is quite complex when you are starting off. I have also learned how to efficiently use Visual Studio Code, which is a code editor and is of great benefit when learning web development.

  • Software Development

    Before entering college, I had limited experience and knowledge of software development and the processes behind it. However, after studying it for some time, it has quickly grown to be one of my favourite modules and I have a basic understanding of the language Java. Java is complex and I still have a lot to learn, however I understand the concepts of data types, relational and logical expressions, loops, methods, arrays and arrays lists and I am able to write simple programs using these concepts. I have also learned how to use Eclipse, which is an Integrated Development Environment. It has made the learning process a lot easier and less frustrating, as your code is compiled as you write it, which means it warns you of any errors you may have before you run the program.

Digital media is transmitted as digital, which invloves digital cables or satellites sending binary signals to devices that translate them into audio, video, graphics, etc. I have learned alot in the digital media module, such as understanding binary code, number bases, compression and digitization and image parameters and optimization.
